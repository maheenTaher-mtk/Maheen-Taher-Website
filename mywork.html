3: RAM Game (Really Awesome Memory Game)

          RAM Game is a colour memory game that me and Jane Mark worked tirelessly on in grade 12. The user is initially shown a sequence of two colours and is then asked to replicate the sequence. With each new round, the size of the sequence increases.

          The premise of this project was to experience paired programming with a partner in order to get a better scope of how coding projects are built. We utilized and built upon our knowledge of GUI and Java swing concepts that we learned in class in order to create a flawless user experience. Our biggest challenge throughout this project was the fact that my laptop was being unable to render the app, which meant that we had to take turns coding on a single laptop. After we had reached our MVP goals (making the project functional), we decided it would be best if I designed assets for the game while Jane handled the coding of our stretch goals. This workflow allowed us to add several different accessibility and customization options for the home screen, the game's colour theme, as well as the background music, meeting most of (if not all) of our stretch goals. In the future, we could add modes based on shapes/images rather than just colours, custom keybinds, as well as the option to increase the number of possible answers (currently only four colours but allowing the user to add more would mean more of a challenge).

4: "I'm Cooked", a recipe generation website

          "I'm Cooked" is a recipe generation website made by me and Max Koenigs that is aimed towards helping cooks of all skill levels find recipes to try! The idea is simple: in your very own virtual 'fridge,' you list all of the ingredients you own at home. You can then generate a list of recipes that each have instructions, all tailored to what YOU have.

          When planning this project, Max and I decided it would be best if he handled the back-end side of our project while I handled the front-end. We used concepts such as arrays/lists, classes, inheritence, and also css styling in order to bring the concept of our site to life. Thankfully, we were able to nail down our MVP goals, which included the generator as well as the aesthetic layout and animations, which were close enough to the vibe that we wanted. Unfortunately, we were unable to add a lot of our stretch goals like recipe filters and misspelling detection. A future goal of ours is creating the "Cookbook Creation" section (another one of our stretch goals), which would allow users to share their own recipes and discover other users' recipes.
